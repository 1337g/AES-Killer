<img src="https://i.imgur.com/Jc9c6lp.png" />

### How to Install
    Download jar file from "https://github.com/EbryxLabs/AES-Killer/releases/download/1.0/AES_Killer.jar" and add in burpsuite

### Requirements
- Burpsuite
- Java


### Tested on
- Burpsuite 1.7.36
- Windows 10
- xubuntu 18.04
- Kali Linux 2018
